After unzipping the folder open your code editor in "test tesk" directory not "test-task"
then open a new terminal and then run "cd ./test-task" and then run "npm install" or "yarn install"
and then you can start the front end by running the command "npm start" or "yarn start"
Also,
Open a new terminal and then run "cd ./server" and then run "npm install" or "yarn install"
and then you can start the back end server by running the command "npm start" or "yarn start"
